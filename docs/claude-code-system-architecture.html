🛡

📚

Multi-Agent System

메인 Opus가 복잡도에 따라 전문 서브에이전트를 스폰하여 병렬로 작업합니다. 모든 에이전트는 공유 품질 기준을 준수합니다.

Complexity-Driven Spawning
ComplexityAgentsCriteriaExample
Trivial0단일 파일, 명확한 수정오타 수정, import 추가
Simple12-3 파일, 한 영역컴포넌트 생성
Moderate2-3UI+API 또는 크로스 영역기능 추가 (프론트+백)
Complex3+풀스택, 아키텍처 변경새 모듈 시스템 구축
